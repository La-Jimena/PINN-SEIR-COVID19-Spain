# PINN-SEIR-COVID19-Spain

Physics-Informed Neural Network (PINN) for the inference of the 
transmission function β(t) and the effective reproduction number Rₜ 
during the first wave of COVID-19 in Spain (March–June 2020).

Developed as part of the Bachelor's Thesis:
**"Modelización multicompartimental y optimización de la cinética del 
SARS-CoV-2 en España: un enfoque basado en sistemas dinámicos y 
aprendizaje informado por la física"**
Universidad Politécnica de Madrid, ETSIAAB, June 2026.

Author: Jimena Blanco Repila
Supervisors: Fernando San José Martínez, José Joaquín Sánchez Álvarez

---

## Description

This repository contains the source code for a SEIR compartmental model 
extended with a time-varying transmission rate β(t), coupled with a 
Physics-Informed Neural Network (PINN) for parameter identification. 
The mobility data from Google Community Mobility Reports is integrated 
into the model following the framework proposed by Nouvellet et al. (2021).

The model is validated on synthetic data and then applied to real 
epidemiological data from the Community of Madrid and the province 
of Toledo.

---

## Repository structure

| Script | Description |
|--------|-------------|
| `generate_data_seir_BetaFunction.py` | Generates 50 synthetic SEIR datasets with multiplicative Gaussian noise (amplitude 0.05). Must be run before the validation script. |
| `PINN_toymodel_validacion.py` | Implements two Monte Carlo validation experiments on synthetic data: robustness against noise and against random initialization seeds. Produces Tables 2.2–2.3 and Figures 2.2–2.3 of the thesis. |
| `PINN_Madrid.py` | Implements the PINN on real epidemiological data from the Community of Madrid and Toledo. Produces Tables 3.1–3.3 and Figures 3.2–3.4 of the thesis. |

## Usage

Run the scripts in the following order:

```bash
# Step 1: Generate synthetic data
python generate_data_seir_BetaFunction.py

# Step 2: Validate the PINN architecture on synthetic data
python PINN_toymodel_validacion.py

# Step 3: Apply the PINN to real data (Madrid and Toledo)
python PINN_Madrid.py
```

The real data file `dataset_SEIR_M_2020-03-01.csv` (incidence and 
mobility for Madrid) is required for Step 3. It is generated by 
`Unificacion.py` from the raw RENAVE and Google Mobility data.
